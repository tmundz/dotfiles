# Content Analysis

Wisdom extraction from any content format using adaptive depth and domain-aware section selection.

## Depth Levels

| Level | Sections | Bullets | Closing Sections |
|-------|----------|---------|-----------------|
| Instant | 1 | 8 | None |
| Fast | 3 | 3 each | None |
| Basic | 3 | 5 each | One-Sentence Takeaway only |
| Full (default) | 5-12 | 3-15 | All three |
| Comprehensive | 10-15 | 8-15 | All + Themes & Connections |

**Default mode:** Full.

## Extraction Workflow

Full workflow: `references/extract-workflow.md`
Detail methodology: `references/extract-wisdom.md`

## Output Tone (Critical)

Write at **Level 3 — conversational**, as if telling a friend over coffee:
- Varied sentence lengths, periods not em-dashes
- No "the speaker noted" or "it was suggested"
- Understated, specific, opinionated
- Each bullet shareable as a standalone insight
- Apply "Would I Tweet This?" test to every bullet

## Section Naming

Name sections like magazine headlines, not category labels:
- "The Part Where He Admits He Was Wrong" not "Self-Reflection"
- "Why Your Intuition Fails in These Situations" not "Decision-Making Insights"
- Kill any section that would exist in any content on this topic

## Closing Sections

- **One-Sentence Takeaway** (15-20 words) — the insight that justifies the whole piece
- **If You Only Have 2 Minutes** — 5-7 cream-of-crop bullets
- **References & Rabbit Holes** — follow-up resources mentioned or implied
- **Themes & Connections** (Comprehensive only) — cross-content patterns
